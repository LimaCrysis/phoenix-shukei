集計アプリ Ver.5.4.0

今回の変更
・Safariで認識されやすいjsPDF標準のダウンロード保存方式へ変更
・ファイル名は「詳細PDF_日付.pdf」
・保存後の案内をiPhoneのSafari画面に合わせて修正
・Safari下部のページメニューから「ダウンロード」を開く手順を表示
・保存済みPDFからLINEへ送ることでURLを付けずPDFだけ送信
・履歴一覧のPDFボタンも同じ保存方式
・運転手管理、自動保存、場所単価、月間管理は維持

LINE送信手順
1. 「詳細PDFを保存」を押します。
2. Safari下部のアドレスバー左側にあるページメニューを押します。
3. 「ダウンロード」を開きます。
4. 「詳細PDF_日付.pdf」を開きます。
5. 共有からLINEを選びます。

GitHub更新方法
index.html、style.css、app.js、README.txt を同じリポジトリへ上書きしてください。
更新後はSafariでページを再読み込みしてください。
