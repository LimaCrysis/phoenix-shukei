集計アプリ Ver.5.5.0

今回の変更
・PDFプレビュー方式へ戻しました
・PDFを開く前に注意画面を表示
・注意画面に「共有 → オプション → PDF」の手順を表示
・「自動」のままだとURLが送られる可能性があることを明記
・「確認してPDFを開く」を押すと標準PDFビューアを開く
・履歴一覧のPDFボタンも同じ注意画面を表示
・ダウンロードフォルダを開く手順は不要
・運転手管理、自動保存、場所単価、月間管理は維持

LINE送信手順
1. 「詳細PDFを開く」を押します。
2. 注意画面で「確認してPDFを開く」を押します。
3. PDF画面で共有ボタンを押します。
4. 画面上部の「オプション」を押します。
5. 送信フォーマットを「PDF」に変更します。
6. LINEを選んで送信します。

GitHub更新方法
index.html、style.css、app.js、README.txt を同じリポジトリへ上書きしてください。
更新後はSafariでページを再読み込みしてください。
