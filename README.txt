集計アプリ Ver.5.1.0

今回の変更
・メインボタン名を「詳細PDFを共有」へ変更
・詳細PDF生成後、iPhone/iPadの共有画面を自動表示
・共有対象はPDFファイルだけ
・共有データにURL、本文、Webページ情報を含めない
・LINEを選んだ場合は「詳細PDF_日付.pdf」を添付ファイルとして送信
・共有画面を閉じた場合は何も保存しない
・PDFファイル共有に未対応の環境ではPDFダウンロードへ自動切り替え
・履歴一覧のPDFボタンも同じPDFファイル共有方式
・運転手管理、自動保存、場所単価、月間管理は維持

注意
iOSやLINEの仕様により共有画面の並び順や送り先をアプリ側で固定することはできません。
共有する内容はPDFファイルだけに設定しています。

GitHub更新方法
index.html、style.css、app.js、README.txt を同じリポジトリへ上書きしてください。
更新後はSafariでページを再読み込みしてください。
