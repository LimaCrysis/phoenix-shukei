集計アプリ Ver.2.2.3

追加機能
・PDF保存ボタンからPDFファイルを直接作成
・単価設定で場所名を変更
・場所を自由に追加
・場所を削除
・追加した場所も運転手ごとに集計

注意
PDF作成時はインターネット接続が必要です。
GitHubへ index.html、style.css、app.js、README.txt を上書きしてください。

修正
・単価設定の「場所を追加」が反応しない不具合を修正
・入力途中の場所名と単価を保持したまま追加可能

修正・改善
・設定画面に「新しい場所名」と「単価」の専用入力欄を追加
・場所名と単価を入力して1回で登録可能
・登録直後に運転手の集計画面へ反映
・同名場所の重複登録を防止
・CSS/JavaScriptへバージョン番号を付け、古いキャッシュが残りにくいよう改善

修正
・追加した場所と単価がメイン画面へ即時反映されない不具合を修正
・追加時に全運転手へ新しい回数欄を自動作成
・古いPWA/ブラウザキャッシュを自動削除
・CSS/JavaScriptのキャッシュ番号を2.2.3へ更新
