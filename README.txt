集計アプリ Ver.3.1.0

今回の変更
・入力内容を保存履歴へ自動保存
・同じ日付の自動保存履歴は1件に自動更新
・運転手名や本数を変更すると履歴も自動更新
・入力が空になった日は自動保存履歴から削除
・「保存」ボタンを削除
・総合計表示を削除
・運転手ごとの合計表示はそのまま維持
・管理画面の月間集計は自動保存履歴から計算
・詳細PDFは従来どおり、日付・運転手・場所・本数・単価・小計・運転手合計を表示

GitHub更新方法
index.html、style.css、app.js、README.txt を同じリポジトリへ上書きしてください。
公開ページが古い場合はSafariで再読み込みしてください。
