集計アプリ Ver.2.3.0

今回の変更
・「単価設定」を「場所単価」へ変更
・場所名と単価の追加は押した時点で自動保存
・登録済みの場所名と単価は変更時に自動保存
・設定画面下の重複した保存ボタンを廃止
・設定画面は「閉じる」だけに整理
・追加した場所をメイン画面へ即時反映
・場所削除をメイン画面へ即時反映
・「場所と単価を追加」ボタンを濃い青＋白文字に変更
・並べ替え、お気に入り機能は追加していません

GitHub更新方法
index.html、style.css、app.js、README.txt を同じリポジトリへ上書きしてください。
公開ページが古い場合はSafariで再読み込みしてください。
