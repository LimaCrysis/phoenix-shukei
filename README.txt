集計アプリ Ver.6.0.0

今回のテーマ
「新しい機能を表へ増やさず、今ある機能を安心して使えるようにする」

今回の変更
・アプリ本体をオフライン利用できるPWA構成へ変更
・一度オンラインで正常に開いた後は、圏外や回線混雑時も入力・自動保存・履歴・管理画面を利用可能
・PDF用ライブラリも初回使用後にキャッシュし、以後はオフラインでもPDF作成可能
・管理画面の中にバックアップと復元を追加
・バックアップには運転手、場所単価、履歴、現在の入力内容を保存
・表画面には新しいボタンを追加していません
・画面下にオフライン状態を小さく表示
・LINE共有前の「オプション → PDF」案内は維持
・既存データの保存キーは変更していないため、これまでのデータを引き継ぎます

初回のオフライン準備
1. 更新後のページをオンライン状態で一度開きます。
2. ページを閉じずに数秒待ちます。
3. 画面下に「オフライン利用の準備完了」と表示されたら準備完了です。
4. PDF機能も一度オンライン状態で開いておくと、PDFライブラリが保存されます。

バックアップ
管理 → データ保護 → バックアップ

復元
管理 → データ保護 → 復元

GitHub更新方法
次のファイルを同じリポジトリへ上書き・追加してください。
・index.html
・style.css
・app.js
・README.txt
・manifest.webmanifest
・service-worker.js

更新後はSafariでページを再読み込みしてください。
