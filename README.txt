・管理画面を2つのボタンで切り替える方式へ変更
・「運転手登録」画面には運転手管理とデータ保護を表示
・「集計」画面には月間集計だけを表示
・管理画面を開いた時は「集計」を最初に表示
・選択していない画面は非表示にし、縦に長くならないよう改善
・運転手管理、自動保存、場所単価、PDF、オフライン、バックアップ機能は維持
