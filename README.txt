集計アプリ Phoenix Ver.2.0 GitHub公開版

1. ZIPを解凍してください。
2. index.html、style.css、app.jsをGitHubのphoenix-shukeiへアップロードしてください。
3. 画面下のCommit changesを押してください。
4. Settings → Pages → Deploy from a branch → main → /(root) → Save。
5. 数分後に表示されるURLをSafariで開いてください。

この版の機能：日付、作業者名、作業者追加・削除、＋／－、作業者合計、総合計、全員リセット。
保存・PDFは次版で追加予定です。
