集計アプリ Ver.5.2.0

今回の変更
・ボタン名を「詳細PDFを開く」へ変更
・詳細PDFを生成後、iPhone/iPadの標準PDFビューアで表示
・PDF生成中は専用画面に「詳細PDFを作成しています…」と表示
・標準PDFビューア右上の共有ボタンからLINEなどへ共有
・Webアプリから一時ファイルを直接共有する方式を廃止
・標準PDFビューアを開けない環境ではPDFダウンロードへ自動切り替え
・履歴一覧のPDFボタンも同じPDFビューア方式
・運転手管理、自動保存、場所単価、月間管理は維持

確認手順
1. 「詳細PDFを開く」を押します。
2. PDFが標準ビューアで開くまで待ちます。
3. 右上の共有ボタンを押します。
4. 共有先にLINEが表示されるか確認します。

GitHub更新方法
index.html、style.css、app.js、README.txt を同じリポジトリへ上書きしてください。
更新後はSafariでページを再読み込みしてください。
