集計アプリ Ver.3.0.0

今回の変更
・画面、履歴、詳細PDFの「回数」をすべて「本数」へ変更
・「○回」を「○本」へ変更
・管理者用の「管理」画面を追加
・保存履歴から月間合計を自動集計
・管理画面に月間合計、総本数、保存日数を表示
・管理画面に運転手別集計を表示
・管理画面に場所別集計を表示
・ランキング、順位、優劣を示す表示はありません
・管理画面の集計内容は詳細PDFへ反映されません
・詳細PDFは日付、運転手、場所、本数、単価、小計、合計のみです

GitHub更新方法
index.html、style.css、app.js、README.txt を同じリポジトリへ上書きしてください。
公開ページが古い場合はSafariで再読み込みしてください。
