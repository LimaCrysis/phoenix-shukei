集計アプリ Ver.4.0.0

今回の変更
・入力画面は選択中の運転手1人だけを表示
・大きな左右ボタンで前後の運転手へ切り替え
・現在位置を「1 / 8」の形式で表示
・名前を入力すると切り替え部分にも即時表示
・運転手追加後は新しい運転手へ自動切り替え
・運転手削除後は残っている前後の運転手へ自動切り替え
・人数が増えても入力画面の縦幅を最小限に維持
・自動保存、履歴、詳細PDF、場所単価、管理画面は維持
・詳細PDFの本数表記を確認・修正

GitHub更新方法
index.html、style.css、app.js、README.txt を同じリポジトリへ上書きしてください。
公開ページが古い場合はSafariで再読み込みしてください。
