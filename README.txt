集計アプリ Ver.5.3.0

今回の変更
・ボタン名を「詳細PDFを保存」へ変更
・PDFを標準ビューアの一時URLで開く方式を廃止
・PDFを実ファイルとしてSafariのダウンロードへ保存
・ファイル名を「詳細PDF_日付.pdf」に固定
・保存後にLINE送信までの短い手順を表示
・保存済みPDFを開いて共有するとURLを付けずPDFだけ送信可能
・履歴一覧のPDFボタンも同じ保存方式
・運転手管理、自動保存、場所単価、月間管理は維持

LINE送信手順
1. 「詳細PDFを保存」を押します。
2. Safari上部のダウンロードボタンを開きます。
3. 「詳細PDF_日付.pdf」を開きます。
4. 共有からLINEを選びます。
5. 「コピーを送信」と表示され、URLなしでPDFだけ送信できます。

GitHub更新方法
index.html、style.css、app.js、README.txt を同じリポジトリへ上書きしてください。
更新後はSafariでページを再読み込みしてください。
